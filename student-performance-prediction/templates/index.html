Predict Your Academic Performance

A simple machine learning powered tool that estimates a student's final score based on study habits, attendance, and a few other factors. Built as a college mini-project using Python, Flask and Scikit-learn.

Try a Prediction

About This Project

This project uses a Linear Regression model trained on student records to predict how well a student might perform in their final exams. It looks at things like study hours, attendance, previous marks, and more to give a quick estimate along with some suggestions for improvement.
